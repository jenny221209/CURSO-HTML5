HTML5

HTML es un lenguaje de etiquetas significa (HyperText MarKup Language)

Y se usa para hacer la estructura de la página web


Etiquetas Principales

!Doctype

Etiqueta que se usa para indicar que la página esta realizada en la versión HTML5

HTML

Dentro de estas etiquetas se debe de poner toda la página web


Etiquetas principales dentro de HTML

head

Head es la cabeza de la página web, Aqui agregamos el titulo de la página(pestaña del navegador),

Enlaces de CSS, Enlaces JavaScript, Etiquetas Meta y demas enlaces

Body

El Body es el cuerpo de la página web, Aqui se agrega todos los componentes de la página.

Footer

El Footer es los pies de la página web, En el ponemos enlaces a redes sociales, Números telefonicos, Direcciones, ETC.


Contenedores

Nav (Menú)

El NAV o menú es donde ponemos los enlaces en navegación de la página web

Header (Cabezera)

El header (Cabezera) es donde podemos poner el menú y/o la portada de la página web

Aside (A un lado)

Este contenedor suele estar a los lados, Izquierda o Derecha. Y se suele usar para menus alternos o públicidad

Section (Secciones)

Este contenedor se usa para meter secciones de contenido

Article (Articulos)

Los articulos son pequeños contenedores que se suelen usar dentro de otros contenedores y se usan

para hacer pequeños articulos o cuadros con contenido

Div (Son divisiones) - No recomendable usar

Antiguamente se solia usar pero ya cayo en des-uso, Ya que los dispositivos moviles

no solian reconocer que parte de la página era.


Etiquetas HTML5

Etiquetas H

Las etiquetas H van desde el valor 1 hasta el 6

El valor menor tiene mas importancia que el valor mayor

El valor menor tiene mas tamaño que el valor mayor

h1 h2 h3 h4 h5 h6


P (Parrafos)

Las P son las etiquetas que se usan para poner texto de tipo parrafos


A (Enlaces)

La etiqueta A se usa para crear enlaces (Se puede usar en cualquier parte)

La propiedad o atributo que enlaza es el href="

LLamamos a páginas con su nombre (Página.html) o a secciones con el # (#seccion1)


IMG (Imagenes)

La etiqueta IMG se usa para enlazar imagenes

La propiedad o atributo que enlaza es src=""


UL (Agrupar listas)

La etiqueta UL se usa para agrupar li (listas)


Li (listas)

La etiqueta LI se usa para hacer listas


BUTTON (Boton)

La etiqueta button para hacer botones
